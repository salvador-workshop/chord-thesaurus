# guitarragrams v2

_Version 1 is at https://github.com/rjsalvadorr/guitarragrams_

## Development

### Installation

```
npm install
```

### Running locally

```
npm run dev
```

Launches parcel (default URL: `http://localhost:1234`)

### Running storybook

```
npm run storybook
```

Launches storybook, for a deep dive into components.
